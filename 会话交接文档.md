## 三、已锁定的核心逻辑（务必遵守）

### 售卖两层模型
- 自然流量(=分账)基线 vs 投流(商业化)版本。**分账 = 自然流量结算，不是独立渠道；分销 ≠ 分账（分销是投流的一种）**。

### 内容上传（3 步）
- **Step1 基础信息**：标题/封面/简介 · 内容类型(短剧/漫剧, 字段联动) · 演员(短剧必填, 漫剧不透传)/导演/编剧 · 作者声明(漫剧AIGC剧锁定"含AI生成内容") · 版权信息 · 备案信息
  - 备案信息含：制作方式/制作形式(横竖屏)/出品年份/制作成本(万)/单集平均时长
  - **制作成本×备案号校验**（触发：内容类型=漫剧 且 无备案号）：成本≥80万→重点备案号；≥30万<80万→普通备案号；<30万→V字头备案号；+ "确保客观真实，如有不实自愿担责"提示
  - 「是否全场景展示」字段：**删除，合并进 step3 售卖信息**
- **Step2 单集信息设置**：纯内容（单集标题30字/封面/视频[上传视频·上传链接]），**无售卖属性**。批量操作下拉三项：批量设置信息 / 批量上传视频 / **批量设置解锁集（本轮决策：归 step2）**
- **Step3 售卖信息**（保存后生成"自然流量·待生效"售卖版本）：
  - 售卖方式：全剧付费 / 单集+全剧付费 / 观看广告解锁（**无 IAAP / 广告解锁+单集付费**）
  - 单集售卖属性：逐集 免费/付费(或广告解锁)；**全剧统一单集价，不支持一集一价**
  - 营销付费面板（仅"单集+全剧付费"出现）：多集优惠包 / 虚拟币充值 **二选一互斥**，最多10个；虚拟币赠送剧豆手动填
  - 推广方式：仅发布(默认) / 分账（内容上传阶段只这两种；完整枚举含投流/分销/授权子账号在售卖版本里配）
  - 场景展示设置：是否P页展示(默认否，关闭后无法再开) / 是否搜索展示(默认是, 含自然搜索+商业化搜索)
  - 生效设置：立即生效 / 预约生效(实际=max{预约时间,审核通过时间}) / 待生效(不自动生效)
  - 预约转换售卖方式：IAA↔IAP，只翻售卖方式其余不变；**生成新版本节点=现版本生效时**；转换时间=原版本预约失效时间=新版本预约生效时间；到点原版本失效→新版本生效；IAA→IAP 新版本走"单集+全剧付费"需设全剧价+单集价；IAP→IAA 原面板作废不带入；原版本转换后保留列表、状态=已失效

### 售卖版本状态机（版本级，5 态 × 按钮显隐）— PRD p10 权威表
- 未生效三态（待生效/已预约生效/已失效）→ 只显 **生效、预约生效**
- 生效中两态（已生效/已预约失效）→ 只显 **失效、预约失效**
- 已有预约的：对应"预约"按钮变**改期/取消**；"立即"按钮=提前触发、覆盖预约
- 查看、编辑常驻；复制（脱离多账号级联树，发布后生成新版本）
- 状态流转：生效→已生效；预约生效→已预约生效→(到点)已生效；失效→已失效；预约失效→已预约失效→(到点)已失效
- 批量（版本tab）：批量生效 / 批量预约生效 / 批量失效 / 批量预约失效

### 剧维度上架（PRD 3.2.3）
- 剧级操作：上架 / 预约上架 / 下架 / 更多(编辑、删除)。**无"绑定投放账户"（那是版本级）**
- 未上架/已下架→显示 上架+预约上架+更多；已上架→显示 下架+更多
- 上架/预约上架按钮**两个位置**：内容页短剧tab列表操作项 + 短剧详情页
- **本轮决策1**：上架/预约上架是剧维度，对短剧A上架 → **短剧A进自然流，所有【已生效】的版本支持售卖**（不是强制所有版本生效；待生效版本不因上架而生效）→ 即"对C端售卖 = 剧已上架 且 版本已生效"
- 前置：内容审核通过 + 至少一个售卖版本；审核中/无版本时置灰；红点提示（编辑售卖版本后提示去上架，点击后红点消失）

### 多账号级联（生效/失效传播）
- 树：版权商 →(同步)子账号 / →(分销)分销商 →(同步)其子账号
- 任一节点操作 → **强制级联到"以它为根的整棵子树"**；只向下、不向上、不向旁系
- 生效、失效都强制级联；上游优先；**上游失效具封禁效力（下游不能自行生效）**
- 复制的版本脱离级联树；分销边（跨主体）级联时给通知
- 见 `售卖版本-多账号传播示意图.svg`

### 投放 ID 链路（方案 A，1:1）
- 剧集id(唯一, 取消复制剧) → 售卖版本id(N) → 产品名×sdpa(1:1) → SDPA/DPA商品 → 授权DSP投放 → 效果归因到售卖版本
- 分账不透传DSP、不生成sdpa；面板内档位(全集/1集/2集)不单独生成sdpa；强出扶持=同剧id下各售卖版本sdpa命名一致
- 见 `投放ID链路-流程图（PRD终版）.svg`
